## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Run json-server in same folder as ranek.json to access API
```
json-server ranek.json
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint 
```
